# Security & Privacy

journeybot builds packing suggestions entirely on-device, doesn't require an account, and syncs journeys only through your own iCloud account. See the full [Privacy Policy](https://journeybot.app/privacy) for details on what data the app touches and how it's handled.

## Reporting a vulnerability

If you've found a security issue in journeybot, please don't open a public issue for it. Instead, email **hello@journeybot.app** with details and, if possible, steps to reproduce. Reports are acknowledged as soon as possible.
